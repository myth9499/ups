  A new version 2.2.0 has just been released
    ftp://xmlsoft.org/
    http://xmlsoft.org/

I have added recently:
  - A FAQ http://xmlsoft.org/FAQ.html
  - An internationalization page
          http://xmlsoft.org/encoding.html

The new version provide the following improvements:
  - applied a lot of portability fixes
  - better encoding support/cleanup and saving
  - the HTML parser now correctly handles encodings
  - added xmlHasProp()
  - fixed a serious problem with &#38;
  - fixed another serious problem in the FTP client code
  - cleanup, bugfixes, etc ...
