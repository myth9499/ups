   They are now available from
     ftp://rpmfind.net/pub/libxml

and the Gnome FTP server.

  Took quite some time but I think it's ready. I have added some
late features since the beta last week:

  - significant speedup by tweaking IS_CHAR like macros and the
    places they are used
  - More interfaces for new I/O functions: xmlNewIOInputStream,
    xmlParserInputBufferCreateIO, xmlCreateIOParserCtxt

  Xpath and the HTML parser module still need some cleanup, but
I expect the current set of API to be stable enough for this release.

  I have upgraded the informations about the changes needed when going
from 1.x to 2.x:
    http://xmlsoft.org/upgrade.html

  I hope this will prove easy to upgrade to this final version,

    enjoy,

Daniel
