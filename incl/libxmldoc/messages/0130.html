On Wed, Sep 15, 1999 at 04:23:47PM -0500, Dao, Trien (STP) wrote:
>
> I am looking for a XML parser for OpenVMS (version 6.2) running on DEC Alpha
> written in C. Do you know of any?

  Well this list is the support list for libxml, an XML parser written
in C. You can fetch the source from ftp://rpmfind.net/pub/veillard
for the last version. It is supposed to be fairly portable, but I have
never heard any report on compiling it on OpenVMS. Having a GNU-like
environment may help in the porting process but should not be mandatory.

  Send us patches if modification are necessary for OpenVMS support,
