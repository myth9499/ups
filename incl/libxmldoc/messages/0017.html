I am interested in maintaining some data in XML and then converting
to HTML for publishing as a Web page.

Microsoft used to have a utility, MSXSL.EXE, that worked from
the command line like this --

How does one do something similar the Linux/libxml/GNOME way?

an equivalent utility out there somewhere, or is this software
