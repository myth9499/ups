Hello everyone

I'm involved in a cross-platform project and we decided to use XML for
some new stuff. Looking for a cross-platform LGPL lib for XML lead
directly to libxml, which we've been using on linux for a while already.
libxml compiles out of the box with cygwin and that's very good. But I
needed to end up with .lib files compatible with MS tools .. after

http://www.ce.berkeley.edu/~doolin/code/libxmlwin32/

I guess I need to edit win32config.h to turn on/off some features? (I
don't need the network stuff, and need to take out some other stuff too)
