errors on msdev 6.0:

_xmlXPtrLocationSetMerge
        libxml2.lib(xpath.obj) : error LNK2001: unresolved external symbol
_xmlXPtrFreeLocationSet
        libxml2.lib(xpath.obj) : error LNK2001: unresolved external symbol
_xmlXPtrRangeToFunction
        .\bin/gjobread.exe : fatal error LNK1120: 3 unresolved externals

after that, gjobread has been builded right.
