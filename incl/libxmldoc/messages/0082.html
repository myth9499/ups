  I have commited quite a bunch of stuff to the gnome-xml CVS
base. Mostly there is added preliminary support for XPath :

  http://www.w3.org/TR/xpath
  XPath is a language for addressing parts of an XML document,
  designed to be used by both XSLT and XPointer.

Most of the tree stuff is implemented, but it need more testing, and
extension to libxml like ID/IDREF detection and lookup. Issues
are listed as TODO in the source ;-) (namespace, id and attributes
axis, some function are not implemented, ...).

I also increased the version to 1.4.0, upgraded the HTML support
(input and output) and rebuilt the documentation.

  I will be on hollidays, unreachable next week, it should not break
stuff but if you use libxml, it's a good idea to check against
the latest version before Friday,

   yours,
