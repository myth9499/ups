http://www.ce.berkeley.edu/~doolin/code/libxmlwin32/

From the README:

libxml2.0 builds. The example distributed does not.

The tar.gz has everything necessary. Winzip will handle it.
There is a directory called `libxmlwin32' that has workspace
and project files. These should build on devstudio 5.

The static libs can be downloaded directly. I might try
and build dlls later.

The buildwin.log file has 39 compiler warnings, most of
which are due to lack of casting or pointer mismatch.
The win32config.h had to get a define for string.h,
and an include for direct.h to handle getcwd().
I fixed 3 or 3 cast problems. Someone smarter than I will
have to get the others.

Enjoy.
