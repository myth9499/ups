 Yet another serious bugfix release !

 It integrates the patches received recently, I also finally followed
Bjorn Reese suggestion and added xmlStrEqual() (and converted the use
of !xmlStrcmp to xmlStrEqual in libxml).
 I think I fixed the nasty bugs people were complaining about in the
XPath predicate evaluation, this looks clean now and the memory leak
has been removed.
 I also removed a nasty bug in the xmlBuildUri() function leading
to problems when using 2 relative filenames, and another serious
bug showing up when using large CData section.

    ftp://xmlsoft.org/

    http://xmlsoft.org/

  enjoy, and as usual if you think I missed a patch, resend :-\
