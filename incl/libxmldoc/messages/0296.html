> 1. Which sections of the above code do you treat as text nodes? It

~/XML -> cat tst2.xml
<?xml version="1.0"?>
<A B="linux is cool"> Gnome is cool too. </A>

version=1.0
standalone=true
  ELEMENT A
    ATTRIBUTE B
      TEXT
      content=linux is cool
    TEXT
    content= ...
~/XML ->
