Ok, short answer: 2.2.1 is a bit harder to build on win32 than 2.1 so it
won't be today.

Long answer: I need to dedicate half a day to fixing build problems on win32.

So fixing them would be a good thing.

After I fix this stuff, I will ftp over to the debian box and diff the whole
mess.

  - a win32 directory for msvc, borland, watcom, etc stuff. This could have the
