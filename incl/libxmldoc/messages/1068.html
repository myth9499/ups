  Yep a new release again, 2.2.6 should be available at the usual places
including:
    http://xmlsoft.org/
    ftp://xmlsoft.org/
    ftp://ftp.gnome.org/pub/GNOME/stable/sources/libxml/

  Mostly integrating the hash tables, adding xpathInternals.h, bugfixes
I hope the hash table change won't break people code, those were stored
in Dtd content. The implementation is now opaque which is a guarantee
that this should not break again anyway :-)
  
 - Added an hash table module, migrated a number of internal structure
   to those
 - Fixed a posteriori validation problems
 - HTTP module cleanups
 - HTML parser improvements (tag errors, script/style handling,
   attribute normalization)
 - coalescing of adjacent text nodes
 - couple of XPath bug fixes, exported the internal API

  thanks again to all the people who contributed, libxml is starting to
be a community effort, this is a good sign for the long term survival
of the project !
