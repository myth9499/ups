   I'm about to release a version 1.5.0, with the following added:

 - added most of the XML validation, it's in an usable state, though
   some parts like ID checks are missing.
 - improved the xpath implementation, still no attributes/ID or
   comparisons but path queries should work
 - added quite a few tests
 - lots of bugfixes
 - this should fix the isinf and isnan not existing on Solaris and
   AIX platforms
 - For RPM users, the 1.5.0 RPM exports both libxml.so.0 and libxml.so.1
   I did some compatibility tests with existing RPMs of gnumeric and
   gnorpm. It should be possible to update RedHat-6.0 setup with those
   RPMs.

  basically I would like to release 1.5 officially at the end of the
week, and i'm seeking some feedback, I have committed the latest version
on the Gnome CVS base and uploaded tar and RPMs at
    ftp://rpmfind.net/pub/veillard

  some early feedback om the developers already using libxml is extemely
welcome before the big push,

   thanks,

Daniel
