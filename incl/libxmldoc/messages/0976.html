  Incorporates all the recent changes discussed on the list.
    - Should work "out of the box" on Windows now
    - Far better XPath support
    - Numerous bug fixes on HTML support
    - Added XPointer support and a testsuite for a resonable subset
    - And the performance problem when using large buffer seems fixed
    - A posteriori validation fixes
    - lots of various bug fixes

  Thanks to everybody who contributed bug reports, ideas and patches !
