I write to inform you that compiled version of libxml release 1.7.1 and 1.7.2 for HP-UX 11, present on your site, have a problem that results in core dumps using free() to release the memory of buffers allocated.

You must recompile the library uncommenting the line:

/* #define NO_DEBUG_MEMORY */

in the source "xmlmemory.h" as posted in the mailing list xml@rufus.w3.org by Veillard (the libxml's author).

Andrea
